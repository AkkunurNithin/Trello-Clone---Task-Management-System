# 📋 Trello Clone - Task Management System

A modern **Trello-inspired Task Management System** built using **HTML5, CSS3, and JavaScript**. TaskFlow helps users organize their daily tasks using a simple Kanban board interface with drag-and-drop functionality, search, dark mode, and local storage support.

---

## 🚀 Live Features

- 📁 Multiple Board Management
- 📌 Create, Rename & Delete Boards
- 📋 Create, Rename & Delete Lists
- ✅ Add, Edit & Delete Tasks
- 📅 Due Date Support
- 🚩 Priority Levels (Low, Medium, High)
- 🔍 Real-time Task Search
- 🎯 Drag & Drop Tasks
- 🌙 Light / Dark Mode
- 💾 Local Storage Support
- 📱 Responsive Design

## ⚙️ How It Works

1. Open the application.
2. Create a new board.
3. Add task lists such as **To Do**, **Doing**, and **Done**.
4. Create tasks with:
   - Title
   - Description
   - Due Date
   - Priority
5. Move tasks using drag and drop.
6. Search tasks instantly.
7. Switch between Light and Dark Mode.
8. All data is automatically saved using Local Storage.
